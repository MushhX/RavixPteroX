# Roadmap

## MVP

- Auth propia (JWT + refresh + rotación)
- Roles/permisos
- Integración Client API básica (listar servidores, acciones de power)
- UI: login + dashboard base

## Siguientes

- WebSockets para consola/telemetría
- Gestión de archivos, backups, DBs
- Plugins avanzados y marketplace real
- Multi-idioma completo + theming
- Billing/tickets/integraciones
