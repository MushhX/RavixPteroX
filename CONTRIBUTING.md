# Contribución

## Reglas

- Mantén backend y frontend desacoplados.
- No uses acceso directo a la base de datos de Pterodactyl.
- Cualquier integración debe usar la API oficial.

## Convenciones

- TypeScript estricto.
- Validación de entradas (schema).
- Endpoints versionados (`/api/v1`).

## Desarrollo

```bash
npm install
npm run dev
```
